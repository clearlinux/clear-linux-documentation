Clear Linux* OS on Proxmox* Virtual Environment¶

This guide explains how to create a new VM in Proxmox VE 6.1-3, install and run Clear Linux OS on as a guest OS.

  • Prerequisites

  • Download the Latest Clear Linux OS Live Server Image

  • Upload Clear Linux OS Live Server Image to Promox Server

  • Create VM on Proxmox

  • Start VM and Install Clear Linux OS on Promox

Prerequisites¶

  • Proxmox VE 6.1-3 server already set up and you have familiarity with how to use it.

Download the Latest Clear Linux OS Live Server Image¶

  1. Visit our Downloads page.

  2. Download the file clear-<release number>-live-server.iso, also called the Clear Linux OS Server.

    Note

    <release-number> is the latest Clear Linux OS auto-numbered release.

Upload Clear Linux OS Live Server Image to Promox Server¶

  1. Connect to your Proxmox server and log into an account with sufficient permission to create and manage VMs.

  2. Under the Server View window, select the local storage. See Figure 1.

  3. On the right window, click Upload.

  4. Set the Content as ISO image. See Figure 2.

  5. Click Select File… and select the Clear Linux OS ISO.

  6. Click Upload.

    The ISO should now appear in the Content list. See Figure 3.

Create VM on Proxmox¶

  1. Under the Server View window, select your Proxmox node. See Figure 4.

  2. On the right window, click Create VM.

  3. In the General tab: | See Figure 5.

    1. Check the Advanced checkbox.

    2. In the Name field, give the VM a name.

  4. In the OS tab: See Figure 6.

    1. Select Use CD/DVD disc image file (iso).

    2. For Storage, select local.

    3. For ISO image, select the Clear Linux OS ISO you uploaded earlier.

    4. Set the Type to Linux.

    5. Set the Version to 5.x - 2.6 kernel.

  5. In the System tab: See Figure 7.

    1. For BIOS, select OVMF (UEFI).

    2. For Storage, select an appropriate location.

    3. For Machine, select q35.

  6. In the Hard Disk tab: See Figure 8.

    1. For Disk size (GiB), set the desired disk size for your VM. A minimum of 4GB is required for Clear Linux OS.

  7. In the CPU tab: See Figure 9.

    1. Set the Type to host.

    2. For the Extra CPU Flags, scroll to the bottom and turn on the aes setting by clicking the + radio button.

  8. In the Memory tab: See Figure 10.

    1. For Memory (MiB), set a desired value.

  9. In the Network tab: See Figure 11.

    1. For Model, select E1000.

  10. In the Confirm tab: See Figure 12.

    1. Confirm the settings.

    2. Click Finish to create the VM. The new VM should appear under the Server View window.

Start VM and Install Clear Linux OS on Promox¶

  1. Under the Server View window, select your newly-created VM. See Figure 13.

  2. On the right window, click Start.

  3. Click Console button to bring up a console and interact with it.

  4. Follow the instructions in the Install Clear Linux* OS from the live server guide starting at the Launch the Clear Linux OS Installer section.
